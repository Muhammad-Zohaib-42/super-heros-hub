# Hero Explorer

A robust React application built to explore and filter data from the Superhero API. This project focuses on high-performance data fetching, type safety, and a clean, professional user interface.

## 🚀 Features

*   **Dynamic Filtering:** Easily drill down into hero statistics and attributes.
*   **Type-Safe Architecture:** Built with **TypeScript** to ensure reliable data handling.
*   **Efficient Data Fetching:** Utilizes **TanStack Query** for optimized caching and state management.
*   **Modern UI:** Styled with **Tailwind CSS** for a professional and clean design.

## 🛠 Tech Stack

*   **Framework:** React (Vite)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **State Management:** TanStack Query (React Query)
*   **API:** [Superhero API](https://superheroapi.com/)
